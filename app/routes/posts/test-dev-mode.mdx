# 開発モードテスト記事

この記事は開発環境でのリアルタイムMarkdown変換をテストするために作成されました。

## テスト項目

1. **Markdownの変換**
   - リストや強調表示が正しく表示される
   - コードブロックが正しく表示される

2. **リアルタイム更新**
   - ファイルを編集して保存すると、ブラウザをリロードするだけで反映される
   - ビルドコマンドを実行する必要がない

## サンプルコード

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet('senju.dev');
```

## まとめ

開発環境でのMarkdown変換が正常に動作しています！
