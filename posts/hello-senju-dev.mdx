# senju.dev ブログはじめました

千住エリアの技術者コミュニティ「senju.dev」の公式ブログを開始しました！

## このブログについて

このブログでは以下のような内容を発信していきます：

- **技術記事**: メンバーによる技術的な知見の共有
- **イベントレポート**: 開催したイベントの振り返りや学び
- **コミュニティ活動**: 新しい取り組みやお知らせ
- **ローカル情報**: 千住エリアの技術系スポット紹介

## 投稿方法

このブログはGitHubでの管理されており、Pull Requestベースで記事を投稿できます。

```bash
# 1. リポジトリをフォーク
# 2. 新しい記事を作成
touch posts/my-article.mdx

# 3. 記事を書く
# 4. Pull Requestを送る
```

## MDXフォーマット

記事はMDX形式で書きます。以下は記事のテンプレートです：

```mdx
---
title: 記事のタイトル
date: 2025-06-18
excerpt: 記事の概要（一覧ページに表示されます）
tags: [タグ1, タグ2]
author: あなたの名前
---

# 見出し1

本文をここに書きます...
```
